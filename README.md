# L-systems interpreter

## Features
* Grammar variations:
    1. Stochastic
    2. Context sensitive
    3. Parametric

## Development
1. `py -m pip install --upgrade build`
2. `py -m build`
3. `pip install -e .`

## Usage
Just run `lsys` to start the program.
